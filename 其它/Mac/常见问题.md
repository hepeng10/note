# 配置环境变量
~/.zshrc 环境变量配置文件，默认是 ~/.bash_profile，由于我们一般会使用 zsh，所以就修改 ~/.zshrc。
修改完成后运行 source ~/.zshrc 使其生效。

# nginx 配置
nginx 目录路径：/usr/local/etc/nginx
配置文件：nginx.conf

# mariadb
MySQL 目录路径：/usr/local/var/mysql
### 启动 mariadb 失败
1. mysql.server start 启动失败，使用 `cat /usr/local/var/mysql/Tirion-Mac.local.err` 查询日志，错误信息为 Plugin 'InnoDB' init function returned error，删除 /usr/local/var/mysql 下的 ib_logfile0，在重新执行 mysql.server start 启动成功。

# node
### egg.js
1. 启动报错：`ERROR 8311 nodejs.SequelizeHostNotFoundError: getaddrinfo ENOTFOUND localhost`。是因为 switch hosts 修改了本地 hosts 文件，将 `127.0.0.1	localhost` 这行删除了。加上这个就行

# HiDPI 修改
* Photoshop 画布是按 retina 分辨率渲染的，所以100%显示时图片大小只有普通屏幕的一半，200%显示才符合正常屏幕的视觉效果，将100%显示的快捷键 command+1 设置到200%上即可解决。
* 1440x900的分辨率不够，想要改成1600x1000，需要使用 [HiDPI 工具](https://github.com/xzhih/one-key-hidpi/blob/master/README-zh.md) 自定义分辨率。

# SourceTree
添加 husky 后提示 env: node: No such file or directory，需要使用命令行来启动。
```c
// 1. 编辑 .zshrc
vi ~/.zshrc

// 2. 添加内容
# 添加 openst 命令用于在终端打开 sourcetree，从而让 sourcetree 支持 husky
alias "openst"="open /Applications/SourceTree.app/Contents/MacOS/Sourcetree"

// 3. 生效
source ~/.zshrc
```
之后就可以在控制台使用 openst 命令启动 sourcetree。或者在 alfred 中使用 `>openst` 来运行。
