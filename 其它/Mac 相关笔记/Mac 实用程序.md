# Mac APP
##### 开发者工具
1. Navicat Premium: 数据库管理
2. Transmit: FTP 工具
3. Charles: 抓包工具
4. Fork: git 可视化管理工具
5. Dash: 各种开发语言文档速查工具
6. iTerm2: 最好的终端
7. Termius: SSH，SFTP 终端
8. Postman: 模拟提交数据（貌似系统自带）
***
##### 系统工具
1. Alfred: 加强型 Spotlight
2. Path Finder: 加强型 Finder（某些原因放弃，使用系统自带的 Finder）
3. Paste: 记录复制的历史
4. iStat Menus: 菜单栏展示系统状态
5. Lemon: 腾讯的系统清洗工具
6. iShot: 截图工具
7. Bartender: 菜单栏增强工具
8. BetterTouchTool: 触摸板、Touchbar 增强工具
9. Paragon NTFS: 让 Mac 读写 NTFS 格式硬盘
10. Android File Transfer: 读取 Android 手机文件
11. Plist Edit Pro: 修改系统 .plist 文件
12. eZip: 压缩解压缩工具
13. Parallels Desktop: 安装 Windows 虚拟机
14. 搜狗输入法
15. QuickRes: 快速切换屏幕分辨率
16. Mos: 修改鼠标滚动方向
17. 超级右键：Finder 右键增强工具
***
##### 实用工具
1. PDF Expert: PDF 阅读器
2. XMind Zen: 思维导图工具
3. Music Tag Editor: 修改音乐标签
4. Downie: 下载 YouTube 等网站视频
5. VideoProc: 录屏，视频格式转换
6. MWeb: Markdown 编辑器
7. Movist Pro: 视频播放器
8. 看图: 腾讯看图软件
9. draw.io: 流程图
10. OmniGraffle: 流程图
