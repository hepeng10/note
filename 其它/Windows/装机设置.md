# 输入法
安装搜狗输入法后，删除微软拼音输入法，再新增一个英文输入法方便在游戏中使用，否则用中文输入法使用键盘行走时会几秒一卡。然后还可以在“高级键设置”中将“在输入语言之间”切换的快捷键改为未分配，就是关闭 alt+shift 切换输入法，只使用 win+空格 切换输入法。

# powerToys
安装 powerToys 后可以使用 alt+空格 触发快捷功能，类似 mac 的 alfred。然后 powerToys 可以安装翻译和 everything 插件。

# 管理员运行的程序
vscode, strokeit, clash 等软件可以右键在兼容性中勾选“以管理员身份运行”。但是就不能开机自动启动了，可以在“任务计划程序”中添加任务进行开机启动，可以在触发里设置延迟15s执行，从而在登录后延迟执行，以便能更稳定的开机运行。

# 虚拟化
windows 家庭版没有 hyper-v, 需要自行开启，否则不能安装 WSL，虚拟机可能也无法使用。
开启方法：https://zhuanlan.zhihu.com/p/577980646
开启 hyper-v 后，如果还不行，运行 `bcdedit /enum | findstr -i hypervisorlaunchtype` 如果是 off，则运行 `bcdedit /set hypervisorlaunchtype Auto` 然后重启。

# gitbash 安装 oh-my-zsh
使用 wsl 和 wsl2 在操作 Windows 系统的时候都很慢，最后还是在 gitbash 上安装 oh-my-zsh 来使用命令行了。
教程：https://juejin.cn/post/7122882640998301733
需要注意的点是，新版的 zsh 安装后运行，会有提问，需要回答后才能使用。
在 gitbash 的 zsh 中使用代理和在 linux 一样，不过有些命令支持还是有点问题，所以对 proxy.sh 进行了修改，可以直接使用 127.0.0.1 作为主机名进行代理。代理文件也在当前目录下。
