# 什么是 Markdown
Markdown 是一种轻量级标记语言，它允许人们使用易读易写的**纯文本格式**编写文档，Markdown文件的后缀名便是`.md`。
是更高级的 txt 文件，支持各种格式、列表、数学公式和 HTML 语法等；相对于 word 文档，Markdown 更轻量级，更高效，**跨平台，易于版本控制**。

# 使用场景
Markdown 被广泛应用于各种场景，包括但不限于：
* 编写技术文档
* 创作博客文章
* 编辑 README 文件
* AI 对话生成的内容
* 编写笔记和待办事项

使用人群主要为技术人员和文字工作者。
目前互联网上很多产品都支持 Markdown 来编写内容，如 GitHub、DeepSeek、CSDN、博客园、知乎、简书、有道笔记等。可以专注于编写内容，不用被格式和排版影响思路。
一开始会有一些学习成本，但是熟悉后在编写标准格式的内容时会更加高效。

# 编辑器
Markdown 本质上是纯文本格式，使用记事本就可以编写，但是要预览具体的效果，就需要使用其他工具。因此，我们通常会选择同时支持编写和预览的编辑器。
经过我自己的测试，目前最好用的编辑器还是 vscode + Markdown Preview Enhanced 插件。可选 Markdown Image 插件来插入图片。

# 语法
## 常用语法
1. 标题
    标题有六级，分别对应 HTML 中的 h1~h6 标签。在 Markdown 中，标题的语法为在文本前添加 `#` 符号，每个 `#` 符号对应一级标题，添加两个 `#` 符号对应二级标题，以此类推。
    ```markdown
    # 一级标题
    ## 二级标题
    ### 三级标题
    #### 四级标题
    ##### 五级标题
    ###### 六级标题
    ```
2. 文本格式
    * 斜体
        使用 `*` 符号来表示，如*斜体文本*。
        ```markdown
        *斜体文本1*
        ```
        **注意：很多教程中会使用 `_` 符号来表示斜体文本，但是 `_` 两边必须有空格才有效果，所以不建议使用。**
    * 加粗
        使用 `**` 符号来表示，如**加粗文本**。
        ```markdown
        **加粗文本1**
        ```
    * 斜体加粗
        使用 `***` 符号来表示，如***斜体加粗文本***。
        ```markdown
        ***斜体加粗文本1***
        ```
    * 高亮
        使用 `==` 符号来表示，如==高亮文本==。
        ```markdown
        ==高亮文本1==
        ```
3. 列表
    * 无序列表
        使用 `*`、`+` 或 `-` 符号来表示。记住一个 `*` 就行。
        ```markdown
        * 列表项1
        * 列表项2
        * 列表项3
        ```
    * 有序列表
        使用数字和点号来表示。
        ```markdown
        1. 列表项1
        2. 列表项2
        3. 列表项3
        ```
    **注意：列表需要注意缩进和换行。**
4. 链接
    使用 `[显示文本](链接URL)` 来表示。 链接可以是本地文件或网络链接。跳转本地 Markdown 文件时支持锚点跳转。
    ```markdown
    [这是一个网络链接](https://www.runoob.com/markdown/md-tutorial.html)
    [这是一个本地链接](./lambda.md#32-引用捕获-)
    [内部跳转](#语法)
    [打开Word文档](./test.docx)
    ```
5. 图片
    使用 `![图片描述](图片URL)` 来表示。图片可以是本地图片或网络图片。
    ```markdown
    ![这是一张本地图片](./assets/md.png)
    ![这是一张网络图片](https://markdown.com.cn/hero.png)
    ```
    
6. 代码
    使用 ` 符号来表示行内代码，通常用来区分一些类似代码的文本；使用 ``` 符号来表示代码块，后面可以指定代码语言。
    ```c
    int main()
    {
        printf("Hello, World!\n");
        return 0;
    }
    ```
7. 分割线
    使用 `---` 或 `***` 来表示。
    ```markdown
    ---
    ***
    ```
8. 引用
    使用 `>` 符号来表示，每个引用的文本前添加 `>` 符号，每个符号后面添加一个空格，然后是引用的文本。
    ```markdown
    > 这是一个引用
    > 这是引用第二行
    >> 这是一个嵌套引用
    ```

## 高级语法
* 待办事项
    使用 `- [ ]` 或 `- [x]` 来表示待办事项和已完成事项。
    ```markdown
    - [ ] 待办事项1
    - [x] 已完成事项1
    - [ ] 待办事项2
    - [ ] 待办事项3
    ```
* 表格
    使用 `|` 符号来分隔不同的单元格，使用 `-` 符号来分隔表头和其他行。
    ```markdown
    | 列1 | 列2 | 列3 |
    | --- | --- | --- |
    | 数据1 | 数据2 | 数据3 |
    | 数据4 | 数据5 | 数据6 |
    ```
* 公式
    公式使用 LaTeX 语法，比较复杂需要单独学习。
    [通用LaTeX 数学公式语法手册](http://www.uinio.com/Math/LaTex/)
    使用 `$` 符号来表示行内公式，使用 `$$` 符号来表示块级公式。
    ```markdown
    $a^2 + b^2 = c^2$
    
    $$
    \sqrt{a^2 + b^2} = c
    $$
    ```
* 图表
    支持绘制流程图、序列图、甘特图等。可以使用 Mermaid 语法来绘制图表。这个就更复杂了，需要单独学习。AI 聊天工具中的回答会有这类流程图。
    [Mermaid 中文网](https://mermaid.nodejs.cn/intro/)
    ```mermaid
    graph TD;
        A-->B;
        A-->C;
        B-->D;
        C-->D;
    ```
* HTML
    可以直接在 Markdown 中使用 HTML 标签来布局。
    ```html
    <div style="text-align: center;">
        <img src="./assets/md.png" alt="这是一张图片" style="width: 100px; height: auto;">
    </div>
    ```
