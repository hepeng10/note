进入 vim 后首先是进入命令模式，vim 会等待输入命令。
i|a|o  进入插入模式，不同之处在于是在光标的哪个位置插入，前|后|光标下插入新行

:  进入编辑模式
:set nu  // 设置行号
:set nonu  // 取消行号
gg  // 光标跳到第一行
G  // 光标跳到最后一行
:123  // 到 123 行
^|0  // 光标移动到行首
$  // 光标移动到行尾
x  // 删除光标所在处字符
nx  // 删除光标所在后 n 个字符
D  // 删除光标所在处到行尾内容
dd  // 删除（剪切）光标所在行
ndd  // 删除（剪切）光标所在行后 n 行字符
dG  // 删除光标所在行到文件末尾的内容
:10,18d  // 删除10到18行
yy  // 复制当前行
nyy  // 复制当前行以下 n 行
p|P  // 粘贴在当前光标所在行下|上
r  // 修改模式，替换当前光标处字符
R  // 持续修改模式
u  // 取消上一步操作
/xxxx  // 搜索 xxxx 字符，按 n 继续往下搜索
:set ic|noic  // 进入 不区分|区分 大小写的搜索模式
:%s/xxx/yyy/g  // 全文替换，将 xxx 替换为 yyy。g 表示直接替换，可以改为 c，在每个替换的时候会询问
:w  // 保存修改
:w /root/another.txt  // 另存为 another.txt
:wq  // 保存退出
ZZ  // 同上，保存退出
:q!  // 不保存强制退出
:wq!  //  强制保存退出，对于没有写权限的文件使用。当然，只有 root 和文件所有者用
