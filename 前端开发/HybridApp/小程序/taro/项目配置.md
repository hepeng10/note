# 静态资源打包
config/index.js 中，配置 url-loader 来确定打包静态资源的位置
```js
mini: {
    imageUrlLoaderOption: {
      // 主包和分包的图片都统一打包到 images 目录中，方便上传 CDN
      // 使用 images/ 表示所有资源打包到根目录下的 images 中，使用 [path] 则会根据源代码中的目录结构打包
      name: 'images/[name]_[contenthash:8].[ext]',
      // cdn 上图片目录。需要将项目打包文件中的 images 目录放到此目录中
      publicPath: 'https://static.jgjapp.com/public/mini/'
    },
}
```
